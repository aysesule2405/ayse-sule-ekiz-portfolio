Whisperwind Grove

Senior Design Capstone · North American University · Spring 2026

TypeScript React 19 Node.js / Express Phaser 3 MongoDB Gemini AI ElevenLabs

Full-stack browser game platform with four handcrafted mini-games, persistent player identity, a shared achievement system, AI-evaluated gameplay moments, and AI-generated guardian voices.

Whisperwind Grove is a TypeScript monorepo built with Vite 7 and React 19 on the frontend, Node.js and Express 5 on the backend, Phaser 3.90 for game scenes, and MongoDB with Mongoose 9 for persistence. The platform hosts four distinct playable worlds: Spirit Drift (arcade catching game), Delivery on the Wind (timed package-delivery with pathing), Spirit Sapling (nurturing game where kindness is evaluated by Gemini 2.0 Flash), and Rise of the Half Moon (moon-phase card-placement strategy with AI-assisted move selection). Every completed game session submits score and state data to a shared backend that evaluates achievements using idempotent badge logic, updates leaderboards, and returns animated unlock toasts to the player. ElevenLabs generates per-guardian voice clips server-side, making each player's companion feel alive. The capstone earned full marks for systems integration, AI implementation, and production-quality architecture.

Obi: Local AI Desktop Assistant

LA Hacks 2026 · Figma UI Challenge Winner

Electron Vite Gemma LLM RAG Pipeline Local AI

Cross-platform AI desktop assistant built in 36 hours. Runs a locally-hosted Gemma LLM with sub-second contextual responses and zero cloud dependency. Won the Figma UI Challenge.

Engineered in a 36-hour sprint at LA Hacks 2026: an Electron and Vite shell hosting a locally-run Gemma LLM connected to a custom RAG pipeline. The result is a desktop assistant that searches your files, notes, and documents using natural language with no internet required. No latency spikes, no data leaving the machine, no cloud subscriptions. The interaction model was designed from the ground up to feel immediate and human rather than transactional. That UX approach won the dedicated Figma UI Challenge track at LA Hacks, with judges recognizing it as production-ready among all competing teams.

Reverie

Full-Stack Web App · Personal Project

React Node.js / Express MongoDB Figma UX Design

A personal mood sanctuary where you curate the feelings that make you, you. Save mood boards, reflections, soundtracks, and emotions in beautifully organized spaces.

Reverie is a full-stack web application built around the idea that your inner world deserves the same care as your outer one. Users create named "spaces" around moods or memories, tagging them as cozy, dreamy, melancholic, joyful, or any other tone. Each space holds images, written reflections, and color palettes. A community feed surfaces public spaces for discovery, while the Timeline view surfaces your emotional history at a glance. The backend is Node.js and Express with a MongoDB data layer. User authentication, space CRUD, community discovery, and a saved collection feed are all implemented. The design was built in Figma first, then translated directly to React.

Ghibli Guardians

Rutgers University HackHERS Hackathon · Safety Companion

React Vite Node.js / Express ElevenLabs REST API Emotional UX

A warm safety companion that helps users assess risky situations, generate emergency messages, and get calming guardian-style guidance when they feel alone or unsure.

Built during Rutgers University's HackHERS Hackathon, Ghibli Guardians combines practical safety flows with a softer character-based interface. The backend uses Node.js and Express to evaluate risk from context like time of day, lighting, environment type, and whether the user is alone. Users can add trusted guardians, store their latest coordinates, and generate personalized emergency messages with Google Maps links. The frontend connects those flows to an interactive React experience with chatbot-style guidance, copy/share actions, music, and ElevenLabs voice playback so the product feels supportive instead of alarming.

OpenStaxAlign

Rice Datathon 2026 · 2nd Place Overall

Python scikit-learn TF-IDF Linear SVM MongoDB

ML pipeline that maps educational content to OpenStax curriculum standards, reaching 75% accuracy on multi-class prediction and placing 2nd overall at Rice Datathon 2026.

Built the full pipeline end-to-end over a datathon weekend: JSON ingestion, TF-IDF vectorization, a Logistic Regression baseline, and a tuned Linear SVM that reached 75% accuracy on multi-class educational standard prediction. Designed the MongoDB schema and built all analytics views that turned raw model outputs into a legible curriculum story. The final presentation framed the data narrative clearly enough to place 2nd out of all competing teams at Rice Datathon 2026.
